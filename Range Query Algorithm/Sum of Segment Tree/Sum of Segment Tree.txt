Write a Program to implement the sum of a Segment Tree.

Input format :
The first line of input is n.

The second line of input is the list of elements.

The third line of input is the start and end index for finding the sum.

Th fourth line of input is the index position and new element to be updated.

Input:

6

1 3 5 7 9 11

1 3

1 10

Output format :
Output:

Sum of values in given range = 15

Updated sum of values in given range = 22

Sample test cases :
