Given an array A[] of size N. Solve Q queries. Find the product in the range [L-1, R-1] under modulo P ( P is Prime). 



Examples:  



Input : 

A[] = {1, 2, 3, 4, 5, 6} 

L = 2, R = 5, P = 229

L-1 = 1, R-1 = 4 

So the range is [1,4] and the elements are 2,3,4,5

Output : 120

Sample test cases :
Input 1 :
5
1 3 8 2 5
313 1 3
Output 1 :
24
